Research Projects

Eye-tracking Study (Tobii)

I analyse gaze behaviour using a Tobii device as part of the HCI-Honours module. The study runs with seven participants and reports heat maps, fixations, and related measures.

  • Supervisor: Prof. Tanya Stott
  • Assessor: Dr Bennie Botha
  • Participants: 7
  • Methods: Heat maps, fixation analysis
  • Stack: Python, pandas, Tobii tooling
  • Outputs: Paper and analysis code

Tractor Driving VR

I built a training-oriented VR simulator for safe tractor operation. The prototype presents realistic vehicle dynamics, seat-gated controls, and a structured evaluation workflow. The technical manual is complete. The project ran to completion in under 8 months.

  • Supervisor: Dr B. Botha
  • Co-assessor: C. Cilliers
  • Tools & languages: C#, Unity, XR Interaction Toolkit, Git, Docker
  • Outputs: Technical manual, user manual, honours report
VRUnityC#XRI
