Software Engineering Projects

In Development

Mazzel

Stovel Banking Web-App

A digital-first banking application tailored for rural and underbanked communities. Designed with simplicity, offline capability, and fintech integrations for mobile devices.

In Development

DocFuse

Open-Source Document Merging Application

DocFuse is an open-source desktop application that allows users to merge multiple documents into a single output. It focuses on clean architecture, strong typing in C++, and a clear user flow for combining files such as reports, assignments, and reference material. The first public release is currently in development (pending).

Not Started

Rubik's Cube Mastery AR

An Augmented Reality (AR) training experience designed to teach and visualise step-by-step cube-solving methods in the user's real environment. It combines puzzle logic with overlayed guidance and interactive spatial feedback, and is currently in the concept phase.

In Development

KIKO

Hand-Tracking, Voice & Mobile Remote Control Interface

A multimodal control prototype that combines hand tracking, voice commands, and a companion mobile phone as a remote. The goal was to explore natural ways to switch modes, adjust settings, and trigger actions without relying solely on a physical controller. Currently in early design and experimentation.
